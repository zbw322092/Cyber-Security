# Elementary Cryptography
## Basic Terminology
• plaintext - P or P/T

• ciphertext-CorC/T

• key–K

• encryption (or enciphering) P C
C = E (K, P)

• decryption (or deciphering) C P

• P=D(K,C)henceP*=D(K,E(K,P))
